# REQ-004-01: fanship 概要

## 概要

fanshipは推し活管理アプリである。推しの登録からイベント・グッズ・支出管理まで、ファン活動を総合的にサポートする。

## 主要機能

### 1. 推し登録

| 機能     | 内容                           |
| -------- | ------------------------------ |
| 対象     | アイドル、キャラクター、声優等 |
| 情報     | 名前、画像、ジャンル、メモ     |
| 複数登録 | 複数の推しを管理可能           |

### 2. イベント管理

| 機能         | 内容                     |
| ------------ | ------------------------ |
| 登録         | ライブ、舞台、リリイベ等 |
| 情報         | 日時、場所、チケット情報 |
| カレンダー   | 予定の可視化             |
| リマインダー | （将来検討）             |

### 3. グッズ管理

| 機能 | 内容                     |
| ---- | ------------------------ |
| 登録 | 所持グッズの記録         |
| 情報 | 名前、画像、購入日、金額 |
| 分類 | 推し別、種類別           |

### 4. 支出記録

| 機能 | 内容                       |
| ---- | -------------------------- |
| 記録 | 推し活にかけた費用         |
| 分類 | イベント、グッズ、交通費等 |
| 集計 | 月別、推し別の支出サマリー |

### 5. 共有機能

| 機能     | 内容                                 |
| -------- | ------------------------------------ |
| 共有対象 | イベント、グッズ等を他ユーザーと共有 |
| 用途     | 同担との情報共有、布教用             |

## 対応デバイス

- PC: フル機能
- スマホ: イベント確認、記録入力
