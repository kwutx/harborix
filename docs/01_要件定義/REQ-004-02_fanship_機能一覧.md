# REQ-004-02: fanship 機能一覧

## 概要

fanship（推し活管理アプリ）の全機能を体系的に整理する。

## 機能一覧

### 1. 推し管理機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| OSH-001 | 推し一覧 | 登録した推しの一覧表示 | 必須 |
| OSH-002 | 推し登録 | 新規推しの追加 | 必須 |
| OSH-003 | 推し編集 | 推し情報の編集 | 必須 |
| OSH-004 | 推し削除 | 推しの削除 | 必須 |
| OSH-005 | 推し画像登録 | プロフィール画像のアップロード | 必須 |
| OSH-006 | ジャンル設定 | アイドル、キャラ、声優等の分類 | 必須 |
| OSH-007 | 推し一括登録 | 複数の推しを一度に登録 | 必須 |

### 2. イベント管理機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| EVT-001 | イベント一覧 | 登録イベントの一覧表示 | 必須 |
| EVT-002 | イベント登録 | ライブ、舞台等の登録 | 必須 |
| EVT-003 | イベント編集 | イベント情報の編集 | 必須 |
| EVT-004 | イベント削除 | イベントの削除 | 必須 |
| EVT-005 | チケット情報登録 | 座席、枚数、金額の記録 | 必須 |
| EVT-006 | イベントカレンダー | カレンダー形式での予定表示 | 必須 |
| EVT-007 | 推し別フィルター | 推し別でイベントを絞り込み | 必須 |
| EVT-008 | リマインダー通知 | イベント前の通知 | 任意 |

### 3. グッズ管理機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| GDS-001 | グッズ一覧 | 所持グッズの一覧表示 | 必須 |
| GDS-002 | グッズ登録 | 新規グッズの追加 | 必須 |
| GDS-003 | グッズ編集 | グッズ情報の編集 | 必須 |
| GDS-004 | グッズ削除 | グッズの削除 | 必須 |
| GDS-005 | グッズ画像登録 | グッズ画像のアップロード | 必須 |
| GDS-006 | 推し別分類 | 推し別でグッズを分類 | 必須 |
| GDS-007 | 種類別分類 | 缶バッジ、アクスタ等で分類 | 必須 |
| GDS-008 | グッズ検索 | 名前、推しでの検索 | 必須 |
| GDS-009 | グッズ一括登録 | 複数のグッズを一度に登録 | 必須 |
