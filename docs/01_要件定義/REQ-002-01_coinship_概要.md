# REQ-002-01: coinship 概要

## 概要

coinshipは家計簿アプリである。パートナーとの共有機能を備える。

## 主要機能

### 1. 口座管理

| 機能     | 内容                 |
| -------- | -------------------- |
| 口座種別 | 銀行、現金、共有口座 |
| 操作     | 追加・編集・削除     |
| 残高管理 | 各口座の残高を追跡   |

### 2. 支払い方法管理

| 機能     | 内容                       |
| -------- | -------------------------- |
| 対応種別 | クレカ、電子マネー、QR決済 |
| 紐付け   | 口座との連携               |

### 3. 収支記録

| 機能     | 内容                 |
| -------- | -------------------- |
| 記録項目 | 金額、カテゴリ、メモ |
| 特殊対応 | ポイント利用の記録   |
| 入力方法 | 手動入力             |

### 4. パートナー共有機能

| 機能     | 内容                         |
| -------- | ---------------------------- |
| 共有範囲 | 特定の口座・記録を共有       |
| 権限     | 閲覧・編集（詳細は後で検討） |

### 5. 集計・可視化

| 機能         | 内容                   |
| ------------ | ---------------------- |
| 月別サマリー | 収入・支出の月次集計   |
| 円グラフ     | カテゴリ別支出の可視化 |
| 残高推移     | 時系列での残高変化     |

### 6. PWA対応

| 機能         | 内容                     |
| ------------ | ------------------------ |
| オフライン   | 基本的な閲覧・入力が可能 |
| インストール | ホーム画面に追加可能     |
| 通知         | （将来検討）             |

## 対応デバイス

- PC: フル機能
- スマホ: PWAで日常的な記録に対応
