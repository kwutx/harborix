# REQ-001-02: harborix基盤 機能一覧

## 概要

harborix基盤の全機能を体系的に整理する。

## 機能一覧

### 1. 認証機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| AUTH-001 | ユーザー登録 | 新規ユーザーアカウントの作成 | 必須 |
| AUTH-002 | ログイン | メールアドレス/パスワードでログイン | 必須 |
| AUTH-003 | ログアウト | セッションの終了 | 必須 |
| AUTH-004 | パスワードリセット | パスワードを忘れた場合の再設定 | 必須 |
| AUTH-005 | SSO連携 | 各アプリへの認証情報共有 | 必須 |
| AUTH-006 | Googleログイン | Googleアカウントでログイン | 必須 |

### 2. ユーザー管理機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| USER-001 | プロフィール表示 | ユーザー情報の表示 | 必須 |
| USER-002 | プロフィール編集 | ユーザー名、アイコン等の変更 | 必須 |
| USER-003 | アカウント削除 | アカウントの完全削除 | 必須 |
| USER-004 | メールアドレス変更 | 登録メールアドレスの変更 | 必須 |
| USER-005 | パスワード変更 | パスワードの変更 | 必須 |

### 3. ダッシュボード機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| DASH-001 | アプリ一覧表示 | 利用可能なアプリの一覧 | 必須 |
| DASH-002 | アプリ起動 | 選択したアプリへ遷移 | 必須 |
| DASH-003 | アプリ並び替え | 表示順序のカスタマイズ | 任意 |
| DASH-004 | アプリ非表示 | 使わないアプリを非表示に | 任意 |

### 4. アプリ管理機能（管理者向け）

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| APP-001 | アプリ登録 | 新規アプリの登録 | 必須 |
| APP-002 | アプリ編集 | アプリ情報の編集 | 必須 |
| APP-003 | アプリ削除 | アプリの削除 | 必須 |
| APP-004 | アプリ公開設定 | 公開/非公開の切り替え | 必須 |

### 5. テーマ機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| THEME-001 | テーマ切り替え | ダーク/ライトモードの切り替え | 必須 |
| THEME-002 | テーマ保存 | ユーザーごとのテーマ設定保存 | 必須 |
| THEME-003 | システム連動 | OS設定に連動（将来検討） | 任意 |

### 6. 共通UI機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| UI-001 | 共通ヘッダー | ロゴ、ナビゲーション、ユーザーメニュー | 必須 |
| UI-002 | 共通フッター | コピーライト、リンク | 任意 |
| UI-003 | 通知表示 | 成功/エラーメッセージの表示 | 必須 |
| UI-004 | ローディング | 読み込み中の表示 | 必須 |
| UI-005 | エラーページ | 404、500等のエラー画面 | 必須 |

## 機能数サマリー

| カテゴリ | 必須 | 任意 | 合計 |
| -------- | ---- | ---- | ---- |
| 認証機能 | 6 | 0 | 6 |
| ユーザー管理機能 | 5 | 0 | 5 |
| ダッシュボード機能 | 2 | 2 | 4 |
| アプリ管理機能 | 4 | 0 | 4 |
| テーマ機能 | 2 | 1 | 3 |
| 共通UI機能 | 4 | 1 | 5 |
| **合計** | **23** | **4** | **27** |

## 備考

- 「任意」は初期リリース後に検討する機能
- アプリ管理機能は開発者（自分）のみが使用する想定
