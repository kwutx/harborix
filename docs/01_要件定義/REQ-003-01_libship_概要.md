# REQ-003-01: libship 概要

## 概要

libshipは読書管理アプリである。本の登録から読書記録まで一般的な機能を提供する。

## 主要機能

### 1. 本の登録

| 機能     | 内容                              |
| -------- | --------------------------------- |
| ISBN検索 | バーコード/ISBN入力で書籍情報取得 |
| 手動登録 | タイトル、著者、出版社など        |
| 表紙画像 | 自動取得または手動アップロード    |

### 2. 読書状態管理

| 状態   | 内容             |
| ------ | ---------------- |
| 未読   | 積読本           |
| 読書中 | 現在読んでいる本 |
| 読了   | 読み終わった本   |

### 3. 読書記録

| 機能   | 内容               |
| ------ | ------------------ |
| 開始日 | 読み始めた日       |
| 終了日 | 読み終わった日     |
| 進捗   | ページ数または割合 |

### 4. 感想・評価

| 機能 | 内容                 |
| ---- | -------------------- |
| 評価 | 5段階評価            |
| 感想 | 自由記述メモ         |
| 引用 | 気になった箇所のメモ |

### 5. 本棚管理

| 機能       | 内容                       |
| ---------- | -------------------------- |
| カテゴリ   | ジャンル・タグで分類       |
| 検索       | タイトル、著者での検索     |
| フィルター | 状態、カテゴリでの絞り込み |

## 対応デバイス

- PC: フル機能
- スマホ: 基本的な閲覧・登録
