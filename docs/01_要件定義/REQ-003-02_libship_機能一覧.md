# REQ-003-02: libship 機能一覧

## 概要

libship（読書管理アプリ）の全機能を体系的に整理する。

## 機能一覧

### 1. 本の登録機能

| ID     | 機能名               | 概要                                | 優先度 |
| ------ | -------------------- | ----------------------------------- | ------ |
| BK-001 | ISBN検索登録         | ISBN/バーコードで書籍情報を自動取得 | 必須   |
| BK-002 | 手動登録             | タイトル、著者等を手動入力で登録    | 必須   |
| BK-003 | 本の編集             | 登録済み本の情報編集                | 必須   |
| BK-004 | 本の削除             | 本の削除                            | 必須   |
| BK-005 | 表紙画像取得         | ISBN検索時に表紙画像を自動取得      | 必須   |
| BK-006 | 表紙画像アップロード | 手動で表紙画像をアップロード        | 任意   |

### 2. 読書状態管理機能

| ID      | 機能名           | 概要                       | 優先度 |
| ------- | ---------------- | -------------------------- | ------ |
| STS-001 | 状態変更         | 未読/読書中/読了の状態変更 | 必須   |
| STS-002 | 状態一覧表示     | 状態別の本の一覧表示       | 必須   |
| STS-003 | 読書中フィルター | 現在読書中の本を表示       | 必須   |
| STS-004 | 積読一覧         | 未読本の一覧表示           | 必須   |
| STS-005 | 読了一覧         | 読了本の一覧表示           | 必須   |

### 3. 読書記録機能

| ID      | 機能名       | 概要                      | 優先度 |
| ------- | ------------ | ------------------------- | ------ |
| REC-001 | 読書開始記録 | 読み始めた日を記録        | 必須   |
| REC-002 | 読書終了記録 | 読み終わった日を記録      | 必須   |
| REC-003 | 進捗記録     | ページ数/割合で進捗を記録 | 必須   |
| REC-004 | 進捗表示     | 現在の読書進捗を表示      | 必須   |
| REC-005 | 読書時間記録 | 読書に費やした時間を記録  | 任意   |

### 4. 感想・評価機能

| ID      | 機能名   | 概要                 | 優先度 |
| ------- | -------- | -------------------- | ------ |
| REV-001 | 評価登録 | 5段階評価の登録      | 必須   |
| REV-002 | 感想登録 | 自由記述の感想メモ   | 必須   |
| REV-003 | 感想編集 | 感想の編集           | 必須   |
| REV-004 | 引用登録 | 気になった箇所のメモ | 必須   |
| REV-005 | 引用一覧 | 本ごとの引用一覧表示 | 必須   |

### 5. 本棚管理機能

| ID      | 機能名             | 概要                       | 優先度 |
| ------- | ------------------ | -------------------------- | ------ |
| SHF-001 | 本棚一覧           | 登録した本の一覧表示       | 必須   |
| SHF-002 | カテゴリ登録       | ジャンル/タグの追加        | 必須   |
| SHF-003 | カテゴリ編集       | カテゴリの編集             | 必須   |
| SHF-004 | カテゴリ削除       | カテゴリの削除             | 必須   |
| SHF-005 | 本へのカテゴリ付与 | 本にカテゴリを設定         | 必須   |
| SHF-006 | キーワード検索     | タイトル、著者で検索       | 必須   |
| SHF-007 | カテゴリフィルター | カテゴリで絞り込み         | 必須   |
| SHF-008 | 状態フィルター     | 読書状態で絞り込み         | 必須   |
| SHF-009 | 並び替え           | 登録日、タイトル、評価順等 | 必須   |

### 6. 統計・集計機能

| ID      | 機能名         | 概要                 | 優先度 |
| ------- | -------------- | -------------------- | ------ |
| STT-001 | 読了数表示     | 月別/年別の読了冊数  | 必須   |
| STT-002 | 読書量グラフ   | 時系列での読書量推移 | 必須   |
| STT-003 | カテゴリ別集計 | ジャンル別の読了数   | 必須   |
| STT-004 | 評価分布       | 評価の分布表示       | 任意   |

### 7. デフォルトカテゴリ

| ID      | 機能名       | 概要                               | 優先度 |
| ------- | ------------ | ---------------------------------- | ------ |
| DEF-001 | 初期カテゴリ | システムデフォルトのカテゴリセット | 必須   |

### 8. UI/テーマ機能

| ID     | 機能名           | 概要                         | 優先度 |
| ------ | ---------------- | ---------------------------- | ------ |
| UI-001 | ダークモード     | ダークテーマへの切り替え     | 必須   |
| UI-002 | ライトモード     | ライトテーマへの切り替え     | 必須   |
| UI-003 | システム設定追従 | OS設定に合わせた自動切替     | 必須   |
| UI-004 | テーマ設定保存   | ユーザーのテーマ設定を保持   | 必須   |

- 「任意」は初期リリース後に検討する機能
- ISBN検索APIは基本設計で決定
