# REQ-002-02: coinship 機能一覧

## 概要

coinship（家計簿アプリ）の全機能を体系的に整理する。

## 機能一覧

### 1. 口座管理機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| ACC-001 | 口座一覧表示 | 登録済み口座の一覧表示 | 必須 |
| ACC-002 | 口座登録 | 新規口座の追加 | 必須 |
| ACC-003 | 口座編集 | 口座情報の変更 | 必須 |
| ACC-004 | 口座削除 | 口座の削除 | 必須 |
| ACC-005 | 残高表示 | 各口座の現在残高表示 | 必須 |
| ACC-006 | 残高調整 | 残高の手動調整 | 必須 |
| ACC-007 | 口座間振替 | 口座間の資金移動 | 必須 |

### 2. 支払い方法管理機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| PAY-001 | 支払い方法一覧 | 登録済み支払い方法の一覧 | 必須 |
| PAY-002 | 支払い方法登録 | クレカ、電子マネー、QR決済の追加 | 必須 |
| PAY-003 | 支払い方法編集 | 支払い方法情報の変更 | 必須 |
| PAY-004 | 支払い方法削除 | 支払い方法の削除 | 必須 |
| PAY-005 | 引落口座紐付け | 支払い方法と口座の連携 | 必須 |

### 3. 収支記録機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| TXN-001 | 取引一覧表示 | 収支記録の一覧表示 | 必須 |
| TXN-002 | 支出登録 | 支出の記録 | 必須 |
| TXN-003 | 収入登録 | 収入の記録 | 必須 |
| TXN-004 | 取引編集 | 記録の修正 | 必須 |
| TXN-005 | 取引削除 | 記録の削除 | 必須 |
| TXN-006 | カテゴリ選択 | 取引のカテゴリ分類 | 必須 |
| TXN-007 | メモ入力 | 取引へのメモ追加 | 必須 |
| TXN-008 | ポイント利用記録 | ポイント使用の記録 | 必須 |
| TXN-009 | 日付指定 | 取引日の指定 | 必須 |
| TXN-010 | 取引検索 | キーワード、カテゴリで検索 | 必須 |
| TXN-011 | 取引フィルター | 期間、口座、カテゴリで絞込 | 必須 |

### 4. カテゴリ管理機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| CAT-001 | カテゴリ一覧 | カテゴリの一覧表示 | 必須 |
| CAT-002 | カテゴリ追加 | カスタムカテゴリの追加 | 必須 |
| CAT-003 | カテゴリ編集 | カテゴリ名、アイコンの変更 | 必須 |
| CAT-004 | カテゴリ削除 | カテゴリの削除 | 必須 |
| CAT-005 | デフォルトカテゴリ | 初期カテゴリセットの提供 | 必須 |

### 5. パートナー共有機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| SHR-001 | パートナー招待 | 共有相手の招待 | 必須 |
| SHR-002 | 招待承諾 | 招待の承諾/拒否 | 必須 |
| SHR-003 | 共有口座設定 | 共有する口座の選択 | 必須 |

| SHR-006 | 共有解除 | パートナー共有の解除 | 必須 |

### 6. 集計・可視化機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| RPT-001 | 月別サマリー | 月次の収入・支出集計 | 必須 |

| RPT-006 | 前月比較 | 前月との比較表示 | 必須 |

### 7. PWA機能

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| PWA-001 | オフライン閲覧 | オフラインでのデータ閲覧 | 必須 |
| PWA-002 | オフライン入力 | オフラインでの取引入力 | 必須 |
| PWA-003 | データ同期 | オンライン復帰時の同期 | 必須 |
| PWA-004 | ホーム画面追加 | アプリとしてインストール | 必須 |
| PWA-005 | プッシュ通知 | リマインダー等の通知 | 任意 |

### 8. UI/テーマ機能（@harborix/ui 共通基盤提供）

※以下はharborix基盤の共通UIライブラリ（@harborix/ui）が提供する機能。coinship固有の実装は不要。

| ID | 機能名 | 概要 | 優先度 |
| -- | ------ | ---- | ------ |
| UI-001 | ダークモード | ダークテーマへの切り替え | 必須 |
| UI-002 | ライトモード | ライトテーマへの切り替え | 必須 |
| UI-003 | システム設定追従 | OS設定に合わせた自動切替 | 必須 |
| UI-004 | テーマ設定保存 | ユーザーのテーマ設定を保持 | 必須 |
