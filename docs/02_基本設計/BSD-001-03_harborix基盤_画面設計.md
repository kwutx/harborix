## 画面詳細

### SCR-001: ログイン

Amplify Authenticatorを使用したログイン画面。

| 要素 | 概要 |
| ---- | ---- |
| Authenticator | Amplify UI提供のログインコンポーネント |
| ロゴ | harborixロゴ |

### SCR-002: ユーザー登録

| 要素 | 概要 |
| ---- | ---- |
| Authenticator | Amplify UI提供の登録コンポーネント |
| ロゴ | harborixロゴ |

### SCR-010: ダッシュボード

| 要素 | 概要 |
| ---- | ---- |
| ヘッダー | ロゴ、ユーザーメニュー、テーマ切替 |
| サイドバー | アプリ一覧ナビゲーション |
| アプリカード | 各アプリへのリンクカード |
| クイックアクセス | 最近使用したアプリ |

### SCR-022: 設定（表示）

| 要素 | 概要 |
| ---- | ---- |
| テーマ設定 | ライト/ダーク/システム |
| アカウント | プロフィールリンク |
| ログアウト | ログアウトボタン |

### SCR-030: アプリ管理一覧（管理者: is_admin=true）

| 要素 | 概要 |
| ---- | ---- |
| アプリ一覧テーブル | 登録済みアプリの一覧 |
| 新規登録ボタン | アプリ登録画面へ |
| 公開/非公開トグル | 公開状態の切り替え |

### SCR-031: アプリ登録/編集（管理者: is_admin=true）

| 要素 | 概要 |
| ---- | ---- |
| アプリ名 | テキスト入力 |
| URL | URL入力 |
| 説明 | テキスト入力 |
| アイコン | アイコン選択 |

## レスポンシブ対応

| 画面幅               | 対応                 |
| -------------------- | -------------------- |
| PC (1024px以上)      | サイドバー表示       |
| タブレット (768px〜) | サイドバー折りたたみ |
| スマホ (〜767px)     | ボトムナビ           |
