## 画面詳細

### SCR-001: ログイン

Amplify Authenticatorを使用したログイン画面。

| 要素 | 概要 |
| ---- | ---- |
| Authenticator | Amplify UI提供のログインコンポーネント |
| ロゴ | harborixロゴ |

### SCR-002: ダッシュボード

| 要素 | 概要 |
| ---- | ---- |
| ヘッダー | ロゴ、ユーザーメニュー、テーマ切替 |
| サイドバー | アプリ一覧ナビゲーション |
| アプリカード | 各アプリへのリンクカード |
| クイックアクセス | 最近使用したアプリ |

### SCR-020: 設定

| 要素 | 概要 |
| ---- | ---- |
| テーマ設定 | ライト/ダーク/システム |
| アカウント | プロフィールリンク |
| ログアウト | ログアウトボタン |

## レスポンシブ対応

| 画面幅               | 対応                 |
| -------------------- | -------------------- |
| PC (1024px以上)      | サイドバー表示       |
| タブレット (768px〜) | サイドバー折りたたみ |
| スマホ (〜767px)     | ボトムナビ           |
