## 画面詳細

### SCR-001: ホーム/ダッシュボード

| 要素 | 概要 |
| ---- | ---- |
| 推しカード | 登録した推しのサムネイル一覧 |
| 直近イベント | 今後1週間のイベント |
| 今月の支出 | 当月の推し活費用合計 |
| 最近追加 | 直近に登録したグッズ |
| フォロー中の更新 | フォローした共有コンテンツの最新更新 |

### SCR-013/033: 一括登録画面

| 要素 | 概要 |
| ---- | ---- |
| 共通設定 | 推し/種類など複数行に共通する項目 |
| 入力行リスト | 動的に追加・削除可能な入力行 |
| +行を追加 | 新しい入力行を追加 |
| 一括保存 | 全ての入力内容をまとめて保存 |

（他の画面も同様に設計時に詳細化）

## レスポンシブ対応

| 画面幅 | 対応 |
| ------ | ---- |
| PC (1024px以上) | サイドバー表示 |
| タブレット (768px〜) | サイドバー折りたたみ |
| スマホ (〜767px) | ボトムナビ |
