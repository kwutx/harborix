## 画面詳細

### SCR-001: ホーム/ダッシュボード

#### 構成要素

| 要素           | 概要                 |
| -------------- | -------------------- |
| 口座別残高     | 各口座の残高サマリー |
| 今月の収支     | 当月の収入・支出合計 |
| カテゴリ別支出 | 円グラフ             |
| 直近の取引     | 最新5件程度          |
| クイック入力   | +ボタン              |

#### アクション

| 操作       | 遷移先/動作 |
| ---------- | ----------- |
| 口座タップ | 口座詳細    |
| 取引タップ | 取引編集    |
| +ボタン    | 取引登録    |

### SCR-021: 取引登録/編集

#### 入力項目

| 項目         | 必須 | 入力形式      |
| ------------ | ---- | ------------- |
| 種別         | ○    | 収入/支出切替 |
| 金額         | ○    | 数値入力      |
| 口座         | ○    | セレクト      |
| カテゴリ     | ○    | セレクト      |
| 日付         | ○    | 日付選択      |
| 支払い方法   | -    | セレクト      |
| メモ         | -    | テキスト      |
| ポイント利用 | -    | 数値入力      |

#### バリデーション

| 項目 | ルール     |
| ---- | ---------- |
| 金額 | 1以上      |
| 日付 | 過去〜今日 |

（他の画面も同様に設計時に詳細化）

## レスポンシブ対応

| 画面幅               | 対応                 |
| -------------------- | -------------------- |
| PC (1024px以上)      | サイドバー表示       |
| タブレット (768px〜) | サイドバー折りたたみ |
| スマホ (〜767px)     | ボトムナビ           |
