## 画面詳細

### SCR-001: ホーム/ダッシュボード

| 要素 | 概要 |
| ---- | ---- |
| 読書中の本 | 現在読んでいる本（最大3冊） |
| 今月の読了 | 当月の読了冊数 |
| 最近の読了 | 直近に読了した本 |
| クイック登録 | +ボタン |

### SCR-011: 本の詳細

| 要素 | 概要 |
| ---- | ---- |
| 書籍情報 | タイトル、著者、出版社、表紙 |
| 読書状態 | 未読/読書中/読了、進捗バー |
| 評価 | 星評価（読了時） |
| 感想タブ | 感想メモ |
| 引用タブ | 引用一覧 |

（他の画面も同様に設計時に詳細化）

## レスポンシブ対応

| 画面幅 | 対応 |
| ------ | ---- |
| PC (1024px以上) | サイドバー表示 |
| タブレット (768px〜) | サイドバー折りたたみ |
| スマホ (〜767px) | ボトムナビ |
