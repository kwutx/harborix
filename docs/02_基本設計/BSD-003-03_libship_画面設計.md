│ 📚 本棚    │  │ │ [表紙] │ │ │ │ [表紙] │ │ │ │ [表紙] │ │ │
│            │  │ └────────┘ │ │ └────────┘ │ │ └────────┘ │ │
│ 📊 統計    │  │ タイトル1  │ │ タイトル2  │ │ タイトル3  │ │
│            │  │ ████░░ 65% │ │ ██░░░░ 30% │ │ █░░░░░ 10% │ │

│            │                                                │
│            │  ─────────────────────────────────────────     │
│            │                                                │

│            │  │  📖 読了: 3冊    📚 読書中: 2冊         │   │
│            │  └────────────────────────────────────────┘   │
│            │                                                │
│            │  最近読了した本                                │
│            │  ┌────────────────────────────────────────┐   │

│            │  │ 📗 本のタイトルB        ★★★★★  2/5    │   │

| 要素 | 概要 |
| ---- | ---- |
| 読書中 | 現在読んでいる本（最大3冊）と進捗バー |
| 今月の読書 | 読了数・読書中の数 |
| 最近読了した本 | 直近の読了本と評価 |
| +ボタン | 本の登録（FAB） |
